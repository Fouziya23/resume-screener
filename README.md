# ATS Resume Scanner

A free, self-built ATS resume scanner — created because every ATS scanning tool online costs money.

## What it does
- Upload your resume (PDF or DOCX) and paste any job description
- Get an ATS compatibility score out of 100
- See which skills you matched and which are missing — grouped by category
- Checks for format issues that break real ATS parsing (tables, text boxes, images)
- Detects job title match, experience years, seniority signals, keyword density
- Gives prioritized, actionable recommendations to improve your score

## Tech Stack
- Python, Flask
- PyPDF2, python-docx (resume parsing)
- Rapidfuzz (fuzzy skill matching)
- Custom skill taxonomy with 13 skill categories
- HTML/CSS (no frameworks)

## Why I built this
ATS scanning tools charge ₹500–₹2000/month. This does the same thing for free.
